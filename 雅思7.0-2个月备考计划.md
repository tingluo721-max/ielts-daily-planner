## 🎯 分项策略

### 👂 听力（目标 7.0+）
- **只听一遍**，对答案后再精听
- 关注**信号词**：however / therefore / but / actually
- 错过就跳过，不要因小失大
- 每次错题标注原因：拼写错误 / 关键词漏听 / 干扰项 / 走神

### 📖 阅读（目标 7.5+）
- 时间分配：**20 + 20 + 20** 三篇文章
- 顺序：略读 → 扫读定位 → 精读答题区
- 按题型专项突破：T/F/NG 是最大失分点
- 答案直接写答题卡（纸笔考试），避免誊写出错

### ✍️ 写作（目标 6.5+）
- **Task 2 分值 = Task 1 × 2**，时间分配 40min : 20min
- Task 1：概述段是关键，趋势词要精准
- Task 2：四段式（引言 + 论点1 + 论点2 + 结论）
- 优先**语法准确**，再追求句式多样
- 精准用词：*significant rise* 不是 *big increase*

### 🗣️ 口语（目标 6.5+）
- 扩展公式：**观点 → 原因 → 例子 → 结果**
- 自然填充词：well / actually / from my perspective
- 流利度 > 高级词汇，清晰 > 复杂
- 每天录音，自己听一遍找问题
